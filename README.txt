Данный модуль предназначен для сохранения и печати отчетов\документов.
Для использования необходимо создать экземпляр класса ReportMaker. Конструктор класса принимает строку - наименование документа (по умолчанию создается без названия).
Метод addText добавляет в текстовый документ необходимый текст в качестве первого аргумента. Текст необходимо передвавать в формате QStringList разбитый на абзацы. Дополнительно можно добавить заголовок текста (второй аргумент) и уровень заголовка <h2-h6> html разметки.
Метод addTable добавляет таблицу на основании контейнера данных (первый аргумент) и списка наименований столбцов таблицы (второй аргумент). Дополнительно можно добавить заголовок таблицы и колличество "таблиц", размещаемых на одном листе.
Метод addImage добавляет картинку, расположенную по адресу imgName. Аналогично устанавливается заголовок и уровень заголовка.

Для создания нового отчета\документа используется метод renewDoc (в качестве аргумента указывается наименование отчета\документа).                                          //only <h1>

Для печати и сохранения файла используются методы save(QString reportName, QString reportPath = "") и print() соответственно.
